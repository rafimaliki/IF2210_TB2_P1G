### Requirement

### Cara menjalankan program

1. Clone repository ini

``
git clone https://github.com/rafimaliki/IF2210_TB2_P1G.git
``

2. Masuk ke directory `bin`

``
cd bin
``

3. Run .jar di folder bin

``
java -jar Harvest-Moon-Combat.jar
``
